• Always Declare Document Type

    <!DOCTYPE html>

  • Always Specify alt, width, and height for Images

    <img src="html5.gif" alt="HTML5" style="width:128px;height:128px">

  • Spaces and Equal Signs

    <link rel="stylesheet" href="styles.css">

  • Avoid Long Code Lines

    • NOT convenient to scroll right and left to read the HTML code.
    • Try to avoid too long code lines.

  • Never Skip the title Element

    • very important for search engine optimization (SEO)

  • Close Empty HTML Elements?

    <meta charset="utf-8" />
    • If you expect XML/XHTML software to access your page, keep the closing slash (/), because it is required in XML and XHTML.

  • Add the lang Attribute

    <html lang="en-us">
    • declare the language of the Web page
    • assist search engines and browsers

  • Meta Data

    <html lang="en-us"> <meta charset="UTF-8">
    • ensure proper interpretation and correct search engine indexing

  • Meta Data

    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    • gives the browser instructions on how to control the page's dimensions and scaling

  • Using Style Sheets

    <link rel="stylesheet" href="styles.css" />

  • Loading JavaScript in HTML

    <script src="myscript.js"> </script>
